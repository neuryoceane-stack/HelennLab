# Dossier des icônes publiques

⚠️ **IMPORTANT** : Déposez vos icônes ICI (dans /public/icons) et non dans /src/assets/icons

## Noms des fichiers requis :

1. **livraison-rapide.svg** (ou .png) - Pour "Livraison rapide"
2. **qualite-visuelle.svg** - Pour "Qualité visuelle premium"
3. **prix-competitifs.svg** - Pour "Prix compétitifs"
4. **performance-optimale.svg** - Pour "Performance optimale"
5. **accompagnement-personnalise.svg** - Pour "Accompagnement personnalisé"
6. **moderne-evolutif.svg** - Pour "Moderne et évolutif"

## Formats acceptés :
- SVG (recommandé pour la qualité)
- PNG (avec fond transparent)
- WebP

## Taille recommandée :
- 64x64px minimum
- Format carré de préférence

Une fois les fichiers déposés ici, ils remplaceront automatiquement les émojis dans les cartes.
Si un fichier n'existe pas, l'émoji d'origine s'affichera en fallback.
