# Helenn Lab - Site Web

Site vitrine professionnel pour Helenn Lab, agence spécialisée dans la création de sites web haut de gamme.

## Stack technique

- **Astro** - Framework web moderne et performant
- **HTML sémantique** - Structure accessible et SEO-friendly
- **CSS moderne** - Modules CSS et styles scoped
- **TypeScript** - Typage strict pour la robustesse

## Structure du projet

```
/
├── public/          # Assets statiques (images, fonts, etc.)
├── src/
│   ├── assets/      # Assets source (images, icônes)
│   ├── components/  # Composants réutilisables
│   ├── layouts/     # Layouts de pages
│   ├── pages/       # Pages du site
│   └── styles/      # Styles globaux et modules CSS
└── astro.config.mjs # Configuration Astro
```

## Commandes

- `npm install` - Installer les dépendances
- `npm run dev` - Démarrer le serveur de développement
- `npm run build` - Construire le site pour la production
- `npm run preview` - Prévisualiser la build de production

## Pages

- `/` - Page principale "Nous" (landing page)
- `/contact` - Page de contact et demande de devis
